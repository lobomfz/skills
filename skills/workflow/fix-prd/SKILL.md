---
name: fix-prd
description: "PRD assumption audit. Use when reviewing a PRD after /write-a-prd or checking for unasked assumptions, missing decisions, or unclear requirements."
disable-model-invocation: true
---

Read the PRD. For each decision in it, ask yourself: did the user tell me this, or did I assume it? List every assumption you made without asking. Then grill the user on them one by one.
